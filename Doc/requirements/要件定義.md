\## 1. 背景・目的



\### 1.1 背景

* ゴミ箱が少ない
* ゴミ箱の場所がわかりずらい
* ポイ捨て問題　

などの環境問題がある



また、日本と海外ではゴミ箱の設置状況に違いがあり、観光客が困る場面も多い。



\### 1.2 目的

利用者同士でゴミ箱を共有することで、

* ポイ捨て防止
* 観光地の環境美化
* 観光客の利便性向上

につなげることを目的とする。



\## ターゲットユーザー

| 区分 | 想定ユーザー | ニーズ |

|---|---|---|

| 投稿者・閲覧者 | 地域利用者、観光客 | ゴミ箱場所の記録・地域利用者との情報共有 |



\## システム構成

* Webブラウザで利用可能
* PC利用を想定
* インターネットの接続環境が必要



\## 3. 機能要件



\### 3.1 機能一覧



| ID   | 機能名      | 概要                    | 認証 |

| ---- | -------- | --------------------- | -- |

| F-01 | 投稿一覧表示   | 投稿されたごみ箱情報を一覧形式で表示する  | 不要 |

| F-02 | 投稿詳細表示   | 選択した投稿の詳細情報を表示する      | 不要 |

| F-03 | 投稿登録     | ごみ箱情報を新規登録する          | 不要 |

| F-04 | 画像アップロード | 投稿時に画像をアップロードし表示する    | 不要 |

| F-05 | コメント投稿   | 投稿に対してコメントを登録する（任意機能） | 不要 |

| F-06 | コメント一覧表示 | 投稿に紐づくコメントを表示する（任意機能） | 不要 |



\---



\### 3.2 投稿一覧表示機能



\#### 概要



登録されたごみ箱情報を一覧形式で表示する。



\#### 表示項目



| 項目名  | 内容             |

| ---- | -------------- |

| 観光地名 | ごみ箱が存在する観光地名   |

| 場所説明 | ごみ箱の設置場所に関する説明 |

| 投稿画像 | 投稿時に登録された画像    |

| 投稿日時 | 投稿データの登録日時     |



\#### 補足



\* 一覧は投稿日時の降順で表示する

\* 投稿件数が増加した場合を考慮し、将来的にページング機能を追加可能な構成とする



\---



\### 3.3 投稿登録機能



\#### 概要



利用者がごみ箱情報を登録できる。



\#### 入力項目



| 項目名  | 必須 | 内容         |

| ---- | -- | ---------- |

| 観光地名 | ○  | 観光地名称      |

| 場所説明 | ○  | ごみ箱の設置場所説明 |

| 投稿画像 | △  | ごみ箱の写真     |

| コメント | △  | 補足情報       |



\#### 入力制御



\* 必須項目未入力時は登録不可とする

\* 画像ファイル以外のアップロードは制限する

\* 最大文字数は別途定義する



\---



\### 3.4 投稿詳細表示機能



\#### 概要



選択した投稿の詳細情報を表示する。



\#### 表示項目



| 項目名    | 内容               |

| ------ | ---------------- |

| 投稿画像   | 投稿された画像          |

| 詳細説明   | ごみ箱の詳細情報         |

| 投稿日時   | 投稿日時             |

| コメント一覧 | 投稿に紐づくコメント（任意機能） |



\---



\### 3.5 コメント機能（任意実装）



\#### 概要



投稿に対してコメントを登録・表示できる。



\#### 想定利用例



\* 「現在も利用可能でした」

\* 「場所が変更されていました」

\* 「夜間は利用不可でした」



\#### 制約



\* 不適切な内容は運用側で削除可能とする（将来的対応）
