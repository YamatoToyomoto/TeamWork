# 1. クラス図

## 1.1 モデル / DbContext / Controller 全体図

```mermaid
classDiagram

    class Post {
        +int Id
        +string PlaceName
        +string LocationText
        +string ImagePath
        +string Comment
        +DateTime CreatedAt
    }

    class ApplicationDbContext {
        +DbSet~Post~ Posts
        +DbSet~Comment~ Comments
    }

    class HomeController {
        +Index() IActionResult
    }

    class PostsController {
        -ApplicationDbContext _db

        +Index() IActionResult
        +Details(int id) IActionResult
        +Create() IActionResult
        +Create(Post post) IActionResult
    }

    ApplicationDbContext o-- Post


    PostsController ..> ApplicationDbContext
    PostsController ..> Post

```

---

## 1.2 クラスの責務

| クラス                    | 種類        | 責務                              |
| ---------------------- | --------- | ------------------------------- |
| `Post`                 | エンティティ    | ごみ箱投稿情報を保持                      |
| `ApplicationDbContext` | DbContext | Entity Framework Core による DB 操作 |
| `HomeController`       | コントローラ    | ホーム画面表示                         |
| `PostsController`      | コントローラ    | 投稿一覧 / 詳細 / 登録処理                |

# 2. 処理フロー

## 2.1 投稿一覧表示

```mermaid
flowchart TD

    Start([GET /Posts])

    Controller[PostsController.Index]

    DB[(Database)]

    GetPosts[投稿一覧取得]

    Render[一覧画面生成]

    Show[一覧画面表示]

    Start --> Controller

    Controller --> GetPosts

    GetPosts --> DB

    DB --> Render

    Render --> Show
```

---

## 2.2 投稿登録

```mermaid
sequenceDiagram

    actor User as ユーザー
    participant Browser as ブラウザ
    participant Controller as PostsController
    participant DB as Database

    User->>Browser: 投稿フォーム入力

    Browser->>Controller: POST /Posts/Create

    Controller->>Controller: ModelState 検証

    alt 入力エラー
        Controller-->>Browser: Create.cshtml 再表示
    else OK
        Controller->>Controller: CreatedAt 設定

        Controller->>DB: INSERT INTO posts

        DB-->>Controller: 保存完了

        Controller-->>Browser: Redirect /Posts
    end
```

---

## 2.3 投稿詳細表示

```mermaid
sequenceDiagram

    actor User as ユーザー
```

# 3. テーブル定義書

## 3.1 ER図

```mermaid
erDiagram
    posts {
        int id PK
        varchar place_name
        varchar location_text
        varchar image_path
        datetime created_at
    }
```

---

## 3.2 テーブル定義

### 3.2.1 posts

| カラム           | 型            | NULL     | 説明      |
| ------------- | ------------ | -------- | ------- |
| id            | integer      | NOT NULL | 投稿ID    |
| place_name    | varchar(100) | NOT NULL | 観光地名    |
| location_text | varchar(100)         | NOT NULL | ごみ箱場所説明 |
| image_path    | varchar(100)         | NULL     | 投稿画像パス  |
| created_at    | datetime     | NOT NULL | 投稿日時    |

* 主キー: `id`
